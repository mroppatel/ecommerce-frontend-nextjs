# Next.js E-commerce Demo

This is a small demo e-commerce site built with Next.js (App Router).

## Features
- Product listing (server-side fetch)
- Product details page
- Cart with add/remove/update quantity (persisted in localStorage)
- Wishlist (persisted in localStorage)
- Checkout page with a basic form

## Setup

1. Install dependencies

```bash
npm install
```

2. Run development server

```bash
npm run dev
```

3. Open http://localhost:3000

## API
This project uses the Fake Store API: https://fakestoreapi.com

## Notes
- To deploy: push to GitHub and connect to Vercel for automatic deployments.
- LocalStorage is used for cart and wishlist persistence.
